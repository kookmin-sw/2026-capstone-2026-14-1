# 운동 정확도 평가표

이 문서는 FitPlus의 MediaPipe 기반 운동 채점이 어떤 조건에서 어느 정도 신뢰할 수 있는지 기록하기 위한 데모 검증표다.

목표는 "잘 되는 것처럼 보임"을 주장하는 것이 아니라, 정상 자세와 오류 자세, 카메라 제한 조건에서 기대 피드백과 실제 결과를 비교해 설명 가능성을 확보하는 것이다.

## 기록 원칙

- 각 운동은 정상 자세, 대표 오류 자세, 카메라 품질 문제를 함께 검증한다.
- 실제 결과는 데모 또는 검증 영상으로 실행한 뒤 점수, 피드백, 채점 보류 여부를 그대로 기록한다.
- 실패한 케이스는 숨기지 않고 제한사항에 남긴다.
- 영상 원본 저장이 아니라 검증 메모와 결과 수치 중심으로 기록한다.

## 공통 판정 기준

| 판정 | 의미 |
|------|------|
| 통과 | 기대 점수/피드백/보류 상태가 실제 결과와 대체로 일치 |
| 부분 통과 | 핵심 판단은 맞지만 점수 범위나 피드백 우선순위가 어긋남 |
| 실패 | 기대와 다른 자세 판단, 잘못된 고득점/저득점, 잘못된 보류 발생 |
| 제한사항 | 현재 카메라/추론/규칙 구조상 안정적으로 처리하기 어려운 조건 |

| ID | 시나리오 | 카메라 조건 | 기대 점수/상태 | 기대 피드백 | 실제 결과 | 판정 | 메모 |
|----|----------|-------------|----------------|-------------|-----------|------|------|
| SQ-01 | 정상 스쿼트 | 정면, 전신 보임, 조명 양호 | 80점 이상 | 주요 경고 없음 | 미기록 | 미기록 | 기본 성공 케이스 |
| SQ-02 | 깊이가 얕은 스쿼트 | 측면, 전신 보임 | 낮은 점수 또는 soft fail | 더 깊이 앉으라는 피드백 | 미기록 | 미기록 | 깊이 판정 확인 |
| SQ-03 | 무릎이 안쪽으로 무너짐 | 정면, 무릎/발목 보임 | 중간 이하 점수 | 무릎 정렬 또는 안쪽 무너짐 피드백 | 미기록 | 미기록 | 정면 view metric 확인 |
| SQ-04 | 상반신만 보임 | 정면, 하체 일부 잘림 | 채점 보류 | 전신이 보이도록 카메라 조정 안내 | 미기록 | 미기록 | gate 동작 확인 |
| SQ-05 | 낮은 조명 | 정면 또는 측면, 조명 부족 | 채점 보류 또는 낮은 신뢰도 | 조명/카메라 위치 조정 안내 | 미기록 | 미기록 | 제한사항 후보 |
