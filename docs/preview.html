Core Features

혼자 운동해도
코치가 옆에 있는 것처럼

FitPlus는 운동 시작부터 실시간 자세 분석, 세션 저장, 기록 기반 성장 리포트까지 자율 운동에 필요한 흐름을 하나의 웹 서비스 안에 담았습니다.

📷

실시간 자세 분석

브라우저 카메라와 MediaPipe Pose를 활용해 관절 landmark를 추적하고, 사용자의 자세 품질을 실시간으로 분석합니다.

🎯

운동별 채점 로직

스쿼트, 푸쉬업, 플랭크별로 반복/시간 판정과 자세 점수, 피드백 규칙을 분리해 운동 특성에 맞게 평가합니다.

🧭

자유 운동 & 배우기

사용자는 단일 운동을 즉시 시작하거나, 단계형 가이드를 따라 운동 방법을 익힌 뒤 코칭 세션으로 진입할 수 있습니다.

🧩

루틴 운동

여러 운동을 묶어 나만의 루틴을 만들고, 단계별 세션 흐름에 따라 운동을 수행할 수 있습니다.

📈

히스토리 분석

세션 결과, 점수, 메트릭 요약, 루틴 기록을 저장하고 과거 운동 변화와 개선 포인트를 확인합니다.

🤖

AI 성장 리포트

누적 운동 기록을 기반으로 약점, 개선 방향, 다음 목표를 요약해주는 개인화 코치 리포트를 제공합니다.

Architecture

실시간 분석은 브라우저에서,
기록과 리포트는 서버에서

카메라 영상과 포즈 추론은 클라이언트 중심으로 처리하고, Express 서버는 인증, 세션 저장, 루틴, 히스토리, AI 리포트 API를 담당합니다.

↓ HTTP / JSON
Express Server Auth · Workout APIs · History · Routine · Quest · AI Report · TTS Proxy
↓
Supabase PostgreSQL User · Session · Set Summary · Metric · Routine · Quest · Tier Data

FitPlus가 강조하는 설계 방향

  • 실시간 pose inference는 서버가 아니라 브라우저에서 수행합니다.
  • 서버는 운동 결과, 세트 요약, 세션 이벤트, 분석용 데이터를 저장합니다.
  • 운동별 특화 규칙은 exercise module로 분리해 확장성을 확보합니다.
  • 점수보다 landmark confidence와 품질 게이트를 우선해 잘못된 피드백을 줄입니다.

Safety & Data

운동 보조 서비스로서
명확한 데이터 처리

FitPlus는 운동 자세 개선을 돕는 보조 서비스입니다. 카메라 권한은 자세 분석과 화면 overlay 표시를 위해 사용되며, 의료 진단·치료·재활 처방을 대체하지 않습니다.

  • 카메라 영상 원본은 서버에 저장하지 않습니다.
  • MediaPipe Pose inference는 클라이언트에서 실행됩니다.
  • 서버에는 운동 결과와 분석에 필요한 요약 데이터가 저장됩니다.
