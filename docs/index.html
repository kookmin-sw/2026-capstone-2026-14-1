FitPlus

웹캠 기반 실시간 AI 운동 코칭 서비스. 별도 웨어러블 기기 없이 사용자의 운동 자세를 분석하고, 반복 횟수·자세 점수·피드백을 제공합니다.

프로젝트 소개

FitPlus는 사용자가 웹캠만으로 스쿼트, 푸쉬업, 플랭크 등의 운동을 수행하면 MediaPipe Pose를 통해 신체 랜드마크를 추출하고, 관절 각도와 자세 안정성을 분석하여 실시간 피드백을 제공하는 웹 운동 코칭 서비스입니다.

이 프로젝트의 핵심 목표는 사용자가 혼자 운동할 때도 자신의 자세가 올바른지 확인하고, 운동 기록을 기반으로 개선 방향을 파악할 수 있게 하는 것입니다.

주요 기능

실시간 자세 분석

브라우저에서 MediaPipe Pose를 사용해 사용자의 관절 위치와 운동 자세를 분석합니다.

운동별 점수화

스쿼트, 푸쉬업, 플랭크별 규칙을 기반으로 반복 횟수, 자세 품질, 안정성을 평가합니다.

운동 기록 관리

운동 세션 결과를 저장하고, 히스토리와 통계를 통해 사용자의 변화를 확인합니다.

AI 성장 리포트

누적 운동 기록을 기반으로 개선점, 약점, 다음 목표를 요약합니다.

기술 스택

Frontend

HTML, CSS, Vanilla JavaScript, EJS, MediaPipe Pose

Backend

Node.js, Express, JWT 인증, Supabase 연동

Data / AI

Supabase PostgreSQL, OpenRouter API, AI 운동 리포트

기대 효과

FitPlus는 사용자가 운동 자세를 스스로 점검하기 어려운 문제를 해결하기 위해 실시간 시각 피드백, 점수화, 기록 기반 분석을 제공합니다. 이를 통해 사용자는 반복적인 운동 수행 과정에서 자신의 자세 문제를 인식하고 점진적으로 개선할 수 있습니다.
