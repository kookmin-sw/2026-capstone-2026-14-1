# 2026-04-10 플랭크 시간 기반 채점 규칙

## 목표

플랭크를 `SIDE` 전용 시간 기반 운동으로 지원한다.

- 자유 운동: 세션 시작 전 목표 시간 직접 설정
- 루틴 운동: `target_value`를 목표 시간(초)으로 사용
- 세션 결과: 최고 연속 hold 시간 저장
- 최종 점수: 자세 80 + 시간 20

## 상태 정의

- `SETUP`: 측면 자세로 진입했지만 아직 안정화 확인 전
- `HOLD`: 연속 안정 프레임 조건을 충족한 유효 hold 구간
- `BREAK`: 정렬 붕괴, 프레임 이탈, 품질 저하로 hold가 끊긴 상태

### 상태 의미 상세

- `SETUP`은 "카메라 준비 단계"가 아니라 "채점 가능한 플랭크 자세로 진입했지만 아직 hold로 인정할 만큼 안정화되지 않은 상태"를 뜻한다.
- 사용자에게 보이는 타이머는 `SETUP` 진입 즉시 시작한다.
- 공식 hold 집계와 점수용 자세 평균은 `HOLD` 구간부터만 인정한다.
- `SETUP`이 길게 유지되는 경우는 의도된 UX가 아니라, 보통 점수 스케일이나 안정화 임계값이 과하게 잡힌 신호로 본다.

## 시간 규칙

- 사용자 타이머는 `SETUP` 진입 즉시 시작한다.
- 공식 결과값은 `best_hold_sec`(세션 내 최고 연속 hold 시간)만 사용한다.
- `BREAK` 이후 다시 자세를 잡아도 이전 hold와 이어 붙이지 않는다.
- 세션 중에는 `현재 구간 시간(current segment)`과 `최고 기록(best hold)`을 함께 표시한다.
- 자유 운동은 목표 시간 달성과 별개로 자동 종료하지 않는다. 사용자가 직접 종료한다.
- 루틴 모드는 `target_value`를 목표 시간(초)로 사용하고, 목표 시간 이상 버티면 해당 세트 완료 조건을 충족한 것으로 본다.

## 점수 규칙

- `posture_score`: 플랭크 최고 hold 구간의 평균 자세 점수
- `time_score = min(100, (best_hold_sec / target_sec) * 100)`
- `final_score = posture_score * 0.8 + time_score * 0.2`

### 시간 점수 해석

- `time_score` 정규화는 초 단위 시간을 0~100 점수로 바꾸는 과정이다.
- 예시:
  - 목표 `30초`, 최고 연속 hold `30초` -> `time_score = 100`
  - 목표 `30초`, 최고 연속 hold `24초` -> `time_score = 80`
  - 목표 `30초`, 최고 연속 hold `15초` -> `time_score = 50`
- 목표 시간을 초과해도 `time_score`는 100에서 캡한다.

## 핵심 메트릭

- `hip_angle`: 몸통 일직선
- `spine_angle`: 몸통 수평 유지
- `shoulder_angle`: 어깨 지지 정렬
- `knee_angle`: 다리 펴기
- `elbow_support_angle`: 팔꿈치가 몸 아래로 과하게 말리지 않는지 확인하는 지표

### 메트릭 정책

- `elbow_support_angle`은 "너무 좁게 접히는 경우만 감점"한다.
- 반대로 팔 각도가 넓어지는 방향은 별도 벌점 대상으로 보지 않는다.
- 플랭크의 `spine_angle`은 수직 대비 기울기 각도이므로, 서 있는 자세처럼 `0도` 근처를 이상적으로 보면 안 된다.
- 플랭크에서는 몸통이 바닥과 평행한 상태가 목표이므로 `spine_angle`의 이상 범위는 수평에 가까운 각도로 해석한다.

### 구현/튜닝 메모

- 초기 튜닝에서 `spine_angle`을 너무 낮은 각도(`0~12도`)로 보아 잘하는 플랭크도 저점 처리되는 문제가 있었다.
- 이후 `spine_angle` 기준을 "몸통 수평 유지"로 재정의하고, `hip_angle` 기준도 다소 완화했다.
- 그래도 실제 샘플에서 플랭크의 품질 점수가 비정상적으로 낮으면, 우선 `breakdown` 기준 총점을 0~100으로 정규화하는 live score 경로를 먼저 점검한다.

## UI 원칙

- 자유 운동 플랭크는 세션 페이지에서 목표 시간 설정 후 시작한다.
- 자유 운동 목록에서 바로 묻지 않고, 플랭크 세션 페이지 안에서 목표 시간을 설정한다.
- 권장 UI는 프리셋 버튼(`15/30/45/60/90초`) + 직접 입력 조합이다.
- 목표 시간이 정해지기 전에는 시작 버튼을 비활성화한다.
- 세션 중에는 아래 값을 동시에 보여준다.
  - 현재 구간 시간
  - 최고 기록
  - 목표 시간
  - 현재 자세 점수
  - 현재 상태(`SETUP`/`HOLD`/`BREAK`)
- 목표 시간 도달 시 자동 종료하지 않는다.

## 자유 운동 / 루틴 공통 원칙

- 자유 운동과 루틴 운동은 같은 점수 체계를 사용한다.
- 차이는 `target_sec`의 출처뿐이다.
  - 자유 운동: 세션 시작 전 사용자가 직접 설정
  - 루틴 운동: `routine_setup.target_value`
- 저장 시 대표 결과값은 항상 `best_hold_sec`를 사용한다.

## 현재 합의된 정책 요약

- 플랭크는 `SIDE`만 지원한다.
- 최고 연속 hold만 기록으로 인정한다.
- `final_score`는 자세 80, 시간 20 비중이다.
- 자유 운동 플랭크는 세션 페이지에서 목표 시간을 정한 뒤 시작한다.
- 목표 시간 달성 여부와 관계없이 자유 운동은 사용자가 직접 종료한다.
